# Corso - Training & Seminar Platform

![Corso Preview](https://raw.githubusercontent.com/MuhammadSohaib-pro/Corso/main/assets/images/corso.png)

A modern, responsive landing page for training and seminar programs built with HTML, CSS, and Bootstrap.

## Features

- **Responsive Design** - Fully responsive layout that works on all devices
- **Interactive Carousel** - Showcase training materials with an image slider
- **Registration Form** - Easy-to-use form for seminar registration
- **Course Summary** - Detailed overview of training programs
- **Key Takeaways Section** - Highlight benefits and learning outcomes
- **Newsletter Subscription** - Email list signup functionality
- **Social Media Integration** - Connect with audience across platforms

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- Google Fonts (Montserrat)

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/MuhammadSohaib-pro/Corso.git
```

2. Navigate to the project directory:

```bash
cd Corso
```

3. Open `index.html` in your browser or use a local development server.
